## Notes

- Every step above ran end-to-end without hitting the network.
- `audit` is new in v0.8 — it scans every installed plugin's
  declared scopes and assigns a risk band based on what each scope
  can reach (see `apps/cli/src/commands/audit.ts` for the weights).
- `status` reads the *real* sandbox policy via
  `SecureSandbox.getPolicy()` — the printed allow-lists are exactly
  what the sandbox would enforce on a live task.
- The MockProvider keeps the chat REPL fully offline. Switch via
  `LLM_PROVIDER=openai` (or `anthropic`, `ollama`) for a real
  backend — tests/real/ has live smoke coverage for each.
